# Solana Stack Pack

Optional stack pack for Solana/Web3 projects.

Primary upstream reference:

- https://github.com/solanabr/solana-claude

This pack should remain optional and should not be part of the default SaaS baseline.

Future additions may include:

- Solana-specific rules
- program security checklist
- Anchor/Pinocchio workflow notes
- wallet integration review
- token/account model review
