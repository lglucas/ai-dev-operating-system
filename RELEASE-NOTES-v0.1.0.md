Initial public release of AI Dev Operating System.

- Project Genesis Wizard from idea to first coding sprint.
- `START-HERE.md` as the first Claude Code entry point.
- `.claude/` runtime structure with agents, commands, rules, skills, and settings.
- Business Plan and Product Brief workflow.

- Documentation layers: Product Brief, Business Plan, Technical Plan, Sprints, Changelog, Session Logs, Knowledge Base, Prototype Lab.

- solo founders;
- small technical teams;
- SaaS builders;
- Claude Code users;
- developers using IDEs such as Antigravity, VS Code, Cursor, Windsurf, or similar;
- builders who want vibe coding without vibe chaos.
