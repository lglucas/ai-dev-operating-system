## What it is

A comprehensive **Chinese-language** Python learning resource teaching programming from beginner to advanced over 100 days. Covers fundamentals, web development (Django), databases, data analysis, machine learning, and team project practices.

## When to consult

- Mandarin-speaking team members onboarding to Python.
- Projects where the dev team prefers Chinese-language learning material.
- Comprehensive beginner-to-intermediate Python curriculum.

## When NOT to consult

- Non-Chinese-speaking teams — content is primarily in Simplified Chinese.
- Already proficient Python developers.
- Need quick reference rather than 100-day curriculum.

## How to use

Reference course — no install. Browse on GitHub.

## Fit signals

- Team includes Chinese-speaking learners.
- Project uses Python and Django, with team needing onboarding.

## Conflicts and overlaps

- Massive scope overlap with other Python learning resources, but unique for its language and depth in Chinese.
