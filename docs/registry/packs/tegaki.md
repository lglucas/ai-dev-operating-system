## What it is

A library that turns any font into animated handwriting — stroke-by-stroke text animation rendered across React, Svelte, Vue, SolidJS, Astro, Web Components, and vanilla JS. No manual path authoring; no native dependencies. Ships with four bundled handwriting fonts (Caveat, Italianno, Tangerine, Parisienne) and an interactive font generator for custom fonts.

## When to install

- Project includes hero sections, marketing pages, or landing pages with a "human / handcrafted" feel.
- Onboarding flows or signature-style branding that benefit from animated text.
- Products with personalization features (e.g. "Hi, [Name]" rendered as handwriting).

## When NOT to install

- Backend-only or CLI projects.
- Performance-sensitive surfaces where animation budget is tight.
- Strict design systems that exclude decorative animation.

## How to install

```bash
npm install tegaki
```

## Fit signals

- Frontend stack is React / Svelte / Vue / SolidJS / Astro.
- Brand identity benefits from warm, handcrafted aesthetics.
- Marketing site or onboarding requires distinctive text moments.

## Conflicts and overlaps

- Niche — no real overlap with other registry packs.

## Notes

Pure frontend library. Lightweight and framework-agnostic. Test on low-end devices before shipping animation-heavy hero sections.
