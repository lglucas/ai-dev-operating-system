## What it is

Community-maintained ecosystem of Claude Code skills, agents, hooks, and harness patterns. A broad catalogue of advanced agentic workflow ideas not found in the official Anthropic repo.

## When to install

- Projects that want advanced agentic patterns (loops, hookification, harness orchestration).
- Teams already comfortable with the basics of Claude Code and looking for power-user tooling.
- Projects exploring multi-agent flows, MCP servers, or hook-driven automation.

## When NOT to install

- First-time Claude Code users who would be overwhelmed.
- Projects that need only the basic `.claude/` setup.
- Compliance-sensitive projects without time to audit each skill individually.

## How to install

Install upstream as a Claude plugin or selectively copy skills with attribution.

```bash
open https://github.com/affaan-m/everything-claude-code
```

Local OS reference already exists at `.claude/skills/external/affaan-everything-claude-code.md`.

## Fit signals
