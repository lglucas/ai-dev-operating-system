## What it is

A free, 9-week online course teaching data engineering fundamentals via hands-on modules: containerization (Docker), workflow orchestration (Kestra), data warehousing (BigQuery), analytics engineering (dbt), batch processing (Spark), and streaming (Kafka). Builds production-ready data pipelines.

## When to consult

- Onboarding data engineers from scratch.
- Team needs structured cohort-style learning (formal cohorts at no cost).
- Project requires Docker + Spark + Kafka + dbt + BigQuery stack.

## When NOT to consult

- Project doesn't involve data engineering.
- Need self-paced reference rather than cohort course.

## How to use

Reference course — clone the repo, follow the modules. Free.
