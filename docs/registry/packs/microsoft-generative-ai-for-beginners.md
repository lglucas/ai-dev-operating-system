## What it is

A 22-lesson course teaching everything needed to start building Generative AI applications. Covers LLMs, prompt engineering, text generation, chat applications, RAG, vector databases, function calling, image generation, fine-tuning, AI agents, security, and UX design — with Python and TypeScript code examples.

## When to consult

- Non-developer founders entering generative AI for the first time.
- Onboarding teams to LLM fundamentals before TECHNICAL-PLAN.md is drafted.
- Before designing prompt strategy or RAG architecture.

## When NOT to consult

- Team already proficient with LLMs and RAG.
- Project doesn't use generative AI.

## How to use

Reference course — no install. Run the Jupyter notebooks locally or browse GitHub.

## Fit signals

- WIZARD Stage 1–4 with team that needs to ramp up on genAI.
- Project includes prompt engineering, RAG, or AI agents.

## Conflicts and overlaps

- Complementary to `microsoft-ai-agents-for-beginners` (more agent-specific) and `microsoft-ai-for-beginners` (broader AI scope including classic ML and CV).
- Pairs with `claude-cookbooks` / `openai-cookbook` (provider-specific).

## Notes

Microsoft authorship biases examples toward Azure OpenAI, but core patterns are provider-agnostic.
