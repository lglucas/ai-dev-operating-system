## What it is

A 12-week, 26-lesson, 52-quiz curriculum teaching classic Machine Learning using Python and Scikit-learn. Covers regression, classification, clustering, NLP, time-series forecasting, and reinforcement learning, with R lesson alternatives and real-world datasets from global cultures.

## When to consult

- Onboarding non-ML developers to classic ML fundamentals.
- Teams entering ML projects without prior background.
- Reference for choosing the right algorithm for a tabular problem.

## When NOT to consult

- Project focuses on deep learning / LLMs (use `microsoft-ai-for-beginners` or `llms-from-scratch`).
- Team already proficient with Scikit-learn.

## How to use

Reference course — no install. Run the Jupyter notebooks locally or browse GitHub.

## Fit signals

- Project involves tabular data, classification, regression, or clustering.
- Team includes non-developers learning ML.

## Conflicts and overlaps

- Explicitly distinct from `microsoft-ai-for-beginners` (this is classic ML; that one is deep learning + symbolic AI).

## Notes

Excellent grounding before tackling deep learning. Quizzes make it suitable for structured team learning.
