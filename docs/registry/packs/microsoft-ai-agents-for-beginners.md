## What it is

A 15+ lesson educational course teaching the fundamentals of building AI agents. Includes written content, video tutorials, Python code samples using Microsoft Agent Framework, and supporting materials covering agentic frameworks, RAG, tool use, multi-agent systems, planning, memory, metacognition, MCP, A2A, and production deployment.

## When to consult

- Onboarding non-developer founders or PMs to "what is an AI agent".
- Before designing the agent layer of a project — sanity-check patterns.
- Teaching teammates the vocabulary of MCP, A2A, RAG, tool calling.

## When NOT to consult

- Team already proficient in agent design — this is beginners-level.
- Project uses non-agentic architecture.

## How to use

Reference course — no install. Browse on GitHub or run the Jupyter notebooks locally.

## Fit signals

- TECHNICAL-PLAN.md mentions agents, multi-agent, MCP, or A2A.
- Team includes non-developers learning AI fundamentals.
- Project uses Microsoft Agent Framework.
