## What it is

A 10-week, 20-lesson curriculum covering foundational data science: SQL, Python, Pandas, data visualization, statistics, ethics, cloud-based ML (Azure). Project-based with quizzes.

## When to consult

- Onboarding non-developer founders or analysts to data science.
- Bridging a team between data analysis and data engineering.

## When NOT to consult

- Need data-engineering depth — use `data-engineering-zoomcamp` instead.
- Already proficient with Pandas + SQL.

## How to use

Reference course — clone or browse on GitHub. Free.

## Fit signals

- Project includes data analysis, dashboards, or BI components.
- Team needs ramp-up on Pandas + SQL + visualization.

## Conflicts and overlaps

- Lighter than `python-data-science-handbook` (broader, less depth).
- Lighter than `data-engineering-zoomcamp` (course is DS-flavored, not DE-flavored).

## Notes

Azure / Microsoft Cloud Advocate authorship biases later cloud lessons toward Azure stack.
