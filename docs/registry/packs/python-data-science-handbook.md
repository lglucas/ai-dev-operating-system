## What it is

The complete text of Jake VanderPlas's O'Reilly book "Python Data Science Handbook" provided as free Jupyter notebooks. Covers IPython, NumPy, Pandas, Matplotlib, and Scikit-Learn — the core data-science Python stack.

## When to consult

- Reference for NumPy/Pandas/Matplotlib idioms.
- Onboarding teams to the Python data-science stack.
- Project includes data analysis or visualization.

## When NOT to consult

- Project doesn't use Python or data science libraries.
- Need deep-learning content (book pre-dates the deep-learning era for ML).

## How to use

Reference book — no install. Browse on `jakevdp.github.io/PythonDataScienceHandbook` or clone for offline.

## Fit signals

- Stack includes NumPy / Pandas / Scikit-Learn.
- Project involves data analysis, visualization, or feature engineering.

## Conflicts and overlaps

- Foundational layer for `microsoft-ml-for-beginners` and `made-with-ml` — read this first if no prior NumPy/Pandas exposure.
