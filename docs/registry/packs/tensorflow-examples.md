## What it is

A historically popular collection of TensorFlow tutorials covering both v1 and v2, with notebook + source-code examples for ML basics through neural network architectures. v1 examples are archived in a separate directory.

## When to consult

- Reading reference ML implementations in TensorFlow.
- Teams already committed to TensorFlow as their framework.

## When NOT to consult

- New projects — most modern ML work has shifted to PyTorch; prefer `llms-from-scratch` or `made-with-ml`.
- Need recent TF 2.x best practices — these examples may be dated.

## How to use

Browse on GitHub or clone for offline. No install needed beyond TensorFlow itself.

## Fit signals

- Existing TensorFlow codebase or team preference for TF.
- Project uses TF Hub or TF Serving in production.

## Conflicts and overlaps

- Outdated relative to PyTorch-first resources like `llms-from-scratch` and `made-with-ml`. Use only when TF is required.
