## What it is

A Claude Code skill focused on UI/UX design intelligence — multi-direction UI exploration, visual prompt structuring, and design decision support before implementation.

## When to install

- Design-led products where UI quality is a differentiator.
- Projects entering the Prototype Lab phase of the WIZARD.
- Teams without a dedicated designer who need structured UI decisions.

## When NOT to install

- Backend-only projects, APIs, infra tooling.
- Projects with a strong existing design system already enforced.

## How to install

```bash
open https://github.com/nextlevelbuilder/ui-ux-pro-max-skill
```

Install upstream as a Claude plugin/skill — do not vendor unless license allows.

## Fit signals

- Project includes UI surfaces (web, mobile, dashboard).
- WIZARD is at the Prototype Lab phase.
- Team wants three-direction UI exploration before committing.

## Conflicts and overlaps

- Complementary to `awesome-design-md` (the latter is more about design-system-as-prompt; this is about exploration workflow).
