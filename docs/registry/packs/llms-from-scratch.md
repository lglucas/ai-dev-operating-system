## What it is

Companion code repository for the book "Build a Large Language Model (From Scratch)" by Sebastian Raschka (Manning, 2024). Step-by-step PyTorch implementations covering text data handling, attention mechanisms, model architecture, pretraining, and finetuning — building a GPT-like LLM end-to-end.

## When to consult

- Team needs to understand LLM internals (not just use them via API).
- ML engineers preparing to fine-tune or train models from scratch.
- Educational use for understanding transformer architecture.

## When NOT to consult

- Project only consumes LLMs via APIs — overkill.
- No PyTorch or deep-learning context in the team.

## How to use

Reference + working code — clone and run the Jupyter notebooks. Companion 17-hour video course and sequel book on reasoning models also exist.

```bash
git clone https://github.com/rasbt/LLMs-from-scratch.git
```

## Fit signals

- Team has PyTorch experience or wants to learn it.
- Project involves fine-tuning, training, or research with custom LLMs.

## Conflicts and overlaps

- Distinct from cookbooks — this is foundational/educational rather than recipe-based.

## Notes

Code is companion to a paid Manning book — buy the book for the full pedagogical experience. Repository alone covers code + appendices but not the explanatory text.
