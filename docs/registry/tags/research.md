# Tag: research

Packs and resources for research-oriented work: replicating papers, literature review, and academic-grounded analysis.

| Resource | Type | Focus |
|----------|------|-------|
| [`google-research`](../packs/google-research.md) | pack | Google Research code monorepo |
| [`feynman-is`](../external-resources/feynman-is.md) | external | AI research agent (papers + cited claims) |
| [`tradingagents`](../packs/tradingagents.md) | pack | Multi-agent LLM trading framework (paper-backed) |

## When this tag is relevant

- WIZARD Stage 6 (Wave 1 research) — strong fit for academic-grounded research.
- Project replicates / extends a specific research paper.
- Citation-heavy work where claim provenance matters (deeptech, healthtech, academic).

## How to choose

- Need code from a specific Google paper → `google-research`.
- Need a research agent that reads papers + cites every claim → `feynman-is`.
- Both can coexist for serious deeptech projects.
