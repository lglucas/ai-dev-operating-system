## When this tag is relevant

- WIZARD Stage 6 (Wave 1 research) — strong fit for academic-grounded research.
- Project replicates / extends a specific research paper.
- Citation-heavy work where claim provenance matters (deeptech, healthtech, academic).

## How to choose

- Need code from a specific Google paper → `google-research`.
- Need a research agent that reads papers + cites every claim → `feynman-is`.
- Both can coexist for serious deeptech projects.
