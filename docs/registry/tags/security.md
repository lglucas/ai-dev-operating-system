## When this tag is relevant

- Project handles user data, payments, authentication, or PII/PHI.
- Project will go public or face external review.
- Compliance audit (SOC 2, ISO 27001, LGPD, GDPR, HIPAA) is on the roadmap.
- Project owns infrastructure (VMs, kubernetes, on-prem).
