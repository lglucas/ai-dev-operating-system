## What it offers

An open-source AI **research agent** that reads papers, searches the web, writes drafts, runs experiments, and cites every claim — all locally on your computer. Multi-agent workflows for deep research, literature review, peer review, code audits, and experiment replication. Integrates AlphaXiv, web search (Gemini/Perplexity), and burst GPU compute via Modal or RunPod.

## When to consult

- Project requires literature review or paper analysis.
- WIZARD Stage 6 (Research Wave 1) — strong fit for academic/research-grounded research.
- Replicating experiments from papers.
- Citation-heavy work where claim provenance matters.

## When NOT to consult

- Quick / casual research — overkill.
- Projects without research components.

## How to use

Visit feynman.is for installation — uses local Docker containers plus optional remote GPU compute. Open source.
