## Adding a new pack

1. Pick a slug — lowercase, hyphenated, matches repo name when possible.
2. Create `packs/<slug>.md` from the template in [`README.md`](README.md#one-pager-template).
3. Add a row to the master table above.
4. Add the slug to every relevant `tags/<tag>.md` file.
5. Cross-link in `UPSTREAM-SOURCES.md` only if the pack also influenced the OS itself.

## Adding a new external resource

1. Create `external-resources/<slug>.md` using the same one-pager template plus `Type: external resource (not installable)`.
2. Add a row to the **External resources** section above.
3. Add the slug to relevant `tags/<tag>.md` files (under an "External resources" subsection if the tag has one).
