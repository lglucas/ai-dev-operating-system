# Philosophy

Vibe coding without an operating system becomes technical debt at AI speed.

This repo treats AI-assisted development as a system composed of:

- rules: constraints the agent must respect;
- agents: specialized roles that reason from different perspectives;
- skills: repeatable workflows with entry and exit conditions;
- commands: compact human interfaces to trigger those workflows;
- sprints: execution containers;
- changelog: release memory;
- session logs: decision memory;
- quality gates: proof that the work is safe enough to merge.

The goal is not to slow down development. The goal is to keep speed from destroying continuity, security, product coherence, and maintainability.
