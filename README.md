# AI Dev Operating System

A practical operating system for AI-assisted product development.

This repository turns vibe coding into a repeatable, auditable workflow for solo founders and small teams using agentic IDEs, Claude Code, Cursor, Codex, Windsurf, and similar tools.

It provides:

- project constitution templates
- modular AI rules
- specialized agents
- reusable skills/workflows
- slash-command specifications
- sprint and changelog lifecycle
- session logs for decision memory
- release and security gates
- attribution tracking for external inspirations

## Core idea

AI-assisted development gets fast before it gets organized. This repo gives you the organization layer: rules, agents, skills, commands, sprint logs, changelogs, and session memory.

## Repository map

```txt
ai-dev-operating-system/
├── docs/                 # Concepts and operating model
├── templates/            # Files to copy into a project
├── rules/                # Modular rules for AI agents and IDEs
├── agents/               # Specialized agent definitions
├── skills/               # Reusable operational workflows
├── commands/             # Slash-command specs
├── examples/             # Example project integrations
└── scripts/              # Helpers for extraction and setup
```

## Start here

1. Read `docs/philosophy.md`.
2. Copy `templates/project/CLAUDE.md` into your target repo.
3. Copy the rules, agents, skills, and commands you want.
4. Start a sprint using `commands/sprint-start.md`.
5. Log important decisions using `templates/session-log/session-log-template.md`.
